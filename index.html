MEUS CONHECIMENTOS

HTML

linguagem de marcação padrão usada para criar a estrutura de páginas da web. Ele define o conteúdo de uma página, como textos, imagens, vídeos e links, organizando-os em uma hierarquia lógica.

CSS

linguagem usada para estilizar a aparência de páginas web escritas em HTML. Ele controla o layout, as cores, as fontes, os espaçamentos e outros aspectos visuais dos elementos, separando a apresentação do conteúdo.

JavaScript

linguagem de programação amplamente usada para adicionar interatividade e dinamismo às páginas web. Ele é executado no navegador do usuário, permitindo manipular elementos HTML, responder a eventos e criar experiências interativas.
